# Inteligência Artificial 

Este repositório reúne atividades, exercícios e implementações desenvolvidas na unidade curricular de **Inteligência Artificial (IA)**. O objetivo é consolidar o aprendizado prático dos conteúdos da disciplina e servir como material de apoio para estudo e revisão.

## Conteúdos abordados

- **História e fundamentos da IA**  
  Introdução à evolução da Inteligência Artificial e seus conceitos principais.

- **Métodos de busca para resolução de problemas**  
  - Busca cega  
  - Busca heurística  
  - Busca competitiva  

- **Representação do conhecimento**  
  Estruturas e técnicas para modelar informações de forma computacional.

- **Aprendizado de máquina**  
  - Aprendizado supervisionado  
  - Aprendizado não-supervisionado  

- **Aplicações de IA**  
  - Processamento de Linguagens Naturais (PLN)  
  - Jogos  
  - Robótica  
  - Mineração de dados  

## Tecnologias utilizadas

- **Python** — principal linguagem para implementação dos algoritmos
- Bibliotecas padrão para manipulação de dados e visualização (dependendo da atividade)
